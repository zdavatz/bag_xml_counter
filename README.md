bag_xml_counter
===============

Counting Limitations Texts in BAG XML from http://galinfo.net

## run like this

  $ ruby ./counter_rexml.rb XMLPublications.zip

## The output will look like this:

  "preparations [de]: 927"
  "preparations [fr]: 927"
  "packs [de]: 23"
  "packs [fr]: 23"
  "itcodes [de]: 727"
  "itcodes [fr]: 727"
